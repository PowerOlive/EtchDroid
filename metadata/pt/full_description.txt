O EtchDroid é uma aplicação sob a licença GPLv3+ para ajudar a gravar imagens em unidades USB.
Use-o para fazer uma unidade USB GNU/Linux inicializável quando o seu portátil estiver morto e estiver no meio do nada.

Características:
- Suporta imagens de disco em bruto (RAW) e a maioria das distribuições GNU/Linux ISOs
- Suporta imagens Raspberry Pi (extraia primeiro o ficheiro ZIP; use um adaptador SD para USB, a ranhura interna de SD não é suportada)
- Suporta a maioria das imagens de disco DMG (beta)
- Sem root - tudo é feito nos limites da API do Android
- Livre como em liberdade

Planeado:
- Gravar a distro comum GNU/Linux em streaming a partir da Internet sem armazená-la no aparelho
- Suporte para ISO de instalação do Windows (provavelmente exigirá root)
- Suporte para flashes de imagem em estilo Unetbootin

O código-fonte encontra-se no GitHub: https://github.com/Depau/EtchDroid
